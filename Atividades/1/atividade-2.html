Glossário de Termos – Desenvolvimento Front-End

Web

A Web é o conjunto de páginas e aplicações acessíveis pela internet, onde usuários podem navegar e interagir com conteúdos.

Página

Uma página é um documento individual da Web, escrito em HTML, que pode conter texto, imagens, links e outros elementos multimídia.

Site

Um site é um conjunto de páginas interligadas, geralmente organizadas sob um mesmo domínio, formando um endereço único na Web.

Front-End

Front-End é a parte do desenvolvimento web responsável pela interface do usuário, ou seja, tudo o que é visível e interativo em um site ou aplicação.

HTML

HTML (HyperText Markup Language) é a linguagem de marcação usada para estruturar o conteúdo das páginas da Web, definindo títulos, parágrafos, listas, links e muito mais.
